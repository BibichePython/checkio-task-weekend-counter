You are given two dates. The task is count all days of rest (saturday and sunday) from first date to second inclusive.

The dates are given as datetime.date (Read about this module). The result is an integer.

Input: Two dates -- from and to inclusive (datetime.date).

Output: An integer.

Example: 

checkio(date(2013, 9, 18), date(2013, 9, 23)) == 2
checkio(date(2013, 1, 1), date(2013, 2, 1)) == 8
checkio(date(2013, 2, 2), date(2013, 2, 3)) == 2

How it is used: Here you will learn how to work with this very useful data type. Also this problem occurred in the software for HR departments or in the calendar apps.
